## 五、管风琴 Würfelspiel 无限背景音乐: 推荐创作方向

### 核心概念: "声景层叠式骰子游戏"

将莫扎特骰子游戏的"预制片段 + 随机组合"理念从单一管风琴扩展为**多层声景**:

```
┌─────────────────────────────────────────────────────┐
│                     输出混音                         │
│                                                      │
│  Layer 3: 自然环境底噪                     [持续淡入淡出] │
│  ├── 雨声 / 森林鸟鸣 / 海浪 / 钟声 / 风声            │
│  ├── 来源: Freesound CC0 采样池                       │
│  └── 切换: 每 5-15 分钟随机渐变到下一个               │
│                                                      │
│  Layer 2: 管风琴低音持续音 (Drone/Pedal)    [缓慢变化] │
│  ├── 使用 JEUX SoundFont 的 Pedal 音栓               │
│  ├── 持续低音: 根据当前和声功能选择根音               │
│  └── 每首华尔兹的和声进行自动更新 drone 音高           │
│                                                      │
│  Layer 1: 骰子游戏主旋律                    [核心引擎] │
│  ├── K.516f 查找表 → 随机选片段                      │
│  ├── 使用 JEUX SoundFont 的各种音栓组合              │
│  └── 每首曲子随机切换: Flute/Principal/Cornet/混合    │
│                                                      │
│  Layer 0: 混响空间                          [全局效果] │
│  ├── 大教堂混响 IR (卷积混响)                        │
│  └── 来源: Freesound CC0 脉冲响应采样                │
└─────────────────────────────────────────────────────┘
```

### 具体实现要点

**1. 管风琴音栓随机化 (Registration Randomization)**

JEUX SoundFont 包含多种音栓组合。每生成一首华尔兹时, 随机选择一种"注册"(registration):

```
registrations = [
    "Flute 8'",           # 柔和长笛
    "Principal 8' + 4'",  # 明亮主音
    "Cornet V",           # 法式短号组合
    "Grand Jeu",          # 全奏华丽
    "Cromorne 8'",        # 簧管独奏
    "Voix Humaine",       # 人声管
]
# 每首华尔兹随机选一种, 连续曲目间自然产生音色变化
```

这让每首华尔兹不仅旋律不同 (骰子游戏), 音色也不同, 极大增加了"不循环"的感受。

**2. 自然声景层 (Ambient Layer)**

从 Freesound 下载一批 CC0 自然采样 (每个 2-10 分钟):

```
ambient_pool = [
    "forest_birds_dawn.wav",      # 清晨森林鸟鸣
    "gentle_rain.wav",            # 轻柔雨声
    "ocean_waves_distant.wav",    # 远处海浪
    "cathedral_ambience.wav",     # 教堂空间底噪
    "wind_through_trees.wav",     # 树林风声
    "creek_flowing.wav",          # 小溪流水
    "thunderstorm_distant.wav",   # 远处雷暴
    "night_crickets.wav",         # 夜晚蟋蟀
]
# 随机选择, 循环播放, 用 5-10 秒 crossfade 切换到下一个
# 音量控制在 -18dB ~ -24dB (不抢管风琴主体)
```

**3. Drone 持续低音层**

管风琴音乐的特色之一是 Pedal 持续低音。在华尔兹的和声骨架基础上:

```
# 根据当前小节的和声功能, 选择 Drone 音
harmony_to_drone = {
    "I":  ["C2", "G2"],   # 主和弦: C + G 持续
    "IV": ["F2", "C3"],   # 下属: F + C
    "V":  ["G2", "D3"],   # 属: G + D
}
# 使用 JEUX 的 Pedal 16' 或 Bourdon 16' 音栓
# 音量较低, 提供和声基础和空间感
```

**4. 时间感知**

让直播有"时间流动"感:

```
# 根据 UTC 时间自动调整氛围
hour = current_utc_hour()
if 5 <= hour < 8:      # 黎明
    ambient = "forest_birds_dawn.wav"
    organ_reg = "Flute 8'"  # 柔和
elif 8 <= hour < 17:    # 白天
    ambient = random.choice(daytime_pool)
    organ_reg = random.choice(bright_registrations)
elif 17 <= hour < 21:   # 黄昏
    ambient = "ocean_waves_distant.wav"
    organ_reg = "Voix Humaine"  # 温暖
else:                    # 夜晚
    ambient = "night_crickets.wav"
    organ_reg = "Cromorne 8'"  # 静谧
```

### 技术路径

在之前的 Rust 架构上扩展:

```
rustysynth (管风琴合成)
    ↓ PCM 流
    + ambient WAV 解码 (rodio/symphonia) → PCM 流
    ↓ 混音
    → FFmpeg stdin → RTMP → YouTube
```

新增的 crate 依赖:

- `symphonia` — 纯 Rust 音频解码, 用于读取 WAV/FLAC 环境音采样
- `rubato` — 音频重采样 (如果采样率不匹配)
- `rustfft` — 可选, 用于频谱分析驱动视觉

### 为什么这个方向特别适合

1. **时代一致性** — JEUX 管风琴 + 莫扎特时代华尔兹 = 18世纪风格统一
2. **无限不循环** — 760万亿旋律组合 × 6+种音栓 × 8种环境音 = 天文级组合数
3. **低资源** — 管风琴音色简单, SoundFont 仅 4MB; 环境音是预录 WAV 循环播放
4. **冥想/氛围市场** — "管风琴 + 自然声景" 正好契合 YouTube 上的冥想/学习/睡眠音乐品类, 这类 24/7 直播频道有大量受众
5. **版权安全** — K.516f 本身是 1787 年作品 (公共领域), JEUX 免费, 环境音选 CC0
